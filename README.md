# Quantum-NS-Ultra
Simulateur IA de fluides quantiques et topologie. Utilise un réseau Physics-ResNet (PINN) pour prédire la vorticité sur des nœuds complexes (Trèfle, 8). Inclut génération vectorisée (10k+ data), contraintes physiques de Navier-Stokes et visualisation 3D interactive. Un pont entre Deep Learning et mécanique des fluides.
